Project Summary
This project is focused on creating a professional portfolio website for Ankit Raj, a Software Engineer and Full-Stack Developer. The website showcases his skills, projects, and experiences, featuring a modern dark theme with glassmorphism effects and smooth animations, all developed using HTML, CSS, and JavaScript without any frameworks.

Project Module Description
The portfolio website consists of the following functional modules:

Navigation Bar: Sticky navigation with smooth scrolling.
Hero Section: Introduction with profile image and tagline.
About Section: Personal background and achievements.
Skills Section: Display of technical skills.
Experience Section: Timeline of professional experiences.
Projects Section: Showcase of featured projects.
Certifications Section: List of relevant certifications.
Education Section: Educational background.
Contact Section: Contact form for inquiries.
Footer: Minimal copyright information.

File Description Inventory

package.json: Contains project metadata and dependencies.
public/assets/profile.jpeg: Profile image displayed in the Hero section.
public/images/: Contains various images used throughout the website.
script.js: JavaScript file that handles animations and interactivity.
style.css: Contains all the styles for the website, including layout and design.
template_config.json: Configuration settings for the templates used in the project.
todo.md: A markdown file for tracking development tasks.
Technology Stack

CSS3: Styling of the website with a focus on responsiveness and aesthetics.
JavaScript: Adding interactivity and animations without using frameworks.
Usage
To set up the project:

Install dependencies: Run the package manager command to install necessary packages.
Build the project: Use the build command specified in the package.json.
Run the project: Start the development server to view the portfolio in a browser.
